# Fry Hard: The Last Fry 🍟

A high-performance, mobile-optimized arcade game where you catch golden fries and trigger high-score "Party Time" events!

## 🎮 How to Play
- **Mobile**: Touch and drag to move the bucket.
- **Desktop**: Use your **Mouse** or **Arrow Keys (A/D)** to move.
- **Goal**: Catch as many golden and baked fries as possible before the 60-second timer runs out.
- **Avoid**: Soggy and burnt fries (they subtract points!).

## ✨ Special Features
- **Party Time**: Catch the rare **Birthday Cake Fry** to trigger 12 seconds of invincibility!
  - **Double Points**: All positive catches count for 2x.
  - **Double Progress**: Move towards a full batch twice as fast.
  - **Blocked Penalties**: Negative fries won't subtract points.
  - **Visuals**: Dynamic countdown timer and accented floating points.
- **Mobile First**: Fully responsive layout with a top-aligned HUD for ergonomic mobile play.
- **Crisp Pixel Art**: Nearest-neighbor scaling for a clean retro aesthetic.

## 🛠️ Tech Stack
- **HTML5 Canvas** for the game engine.
- **Vanilla JavaScript** (No external dependencies).
- **CSS3** with Glassmorphism and dynamic animations.

## 🚀 Deployment
This project is ready for deployment to **Vercel**, **Netlify**, or **GitHub Pages**. Just drop the files in and go!
